Atividade 2 - Cap. 8

Sistemas para internet - FIAP

Desenvolver um formulário com HTML e CSS para cadastro de um ponto turístico.

Para o desenvolvimento, além de dominar HTML e CSS, é importante que uma breve pesquisa seja realizada para identificar quais são os campos necessários para o cadastro de um ponto turístico. Ex.: nome, localização, valor de ingresso etc.

Desenvolvi um formulário para cadastro de um ponto turístico, explorando todo o conteúdo apresentado até o momento no curso. No formulário utilizei:

  • label
  • input text
  • input data
  • input number
  • input telefone
  • input email
  • input url
  • input list/datalist
  • input checkbox
  • fieldset
  • legend
  • time
  • select option
  • textarea
  • input range
  • input file
  • input submit
  • input reset

Na elaboração do layout utillizei display flex, e formatei os textos e labels em "vw" para deixar o cadastro responsivo.

Acesse o formulário clicando Click no menu "cadastro", diminua a tela ou acesse a o fomulário de desktop e de celularfaça um cadastro e avalie minha criatividade e meu desenvolvimento.
